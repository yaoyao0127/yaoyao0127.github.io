  1. BBC工作人员认为现代浏览器应该支持的功能
  • document.querySelector()/document.querySelectorAll()
  • window.addEventListener()
  • 存储API(localStorage和sessionStorage)
  1. jquery可以支持一些古老的浏览器，如IE8或早期版本。
  2. IE9是现代浏览器体验的基础平台。
  3. JS对象拥有属性和方法，属性包含对象的数据，方法是作用在数据上的函数。
  4. JS极少有自修改对象，Array对象的push()/pop()方法是特例。Array对象的值是作为其属性来存储的。
  5. JS语言特性一：引用和赋值
    JS中有两种把数据保存到变量中的方法，赋值和引用。
  • 当保存的是字符串、数字、布尔值、null和undefined这样的原始值时，采用的是赋值的方法，也就是直接把这些原始值复制到变量中。注意数组不是原始值，而字符串是原始值。
  • 当保存的不是原始值时，就采用引用的方法，也就是把指向对象内存位置的指针保存在变量中。此时，修改原对象，变量也会受到影响。
    如下的代码1中，b中保存的是a的复制值，而不是指针。代码2中，newArr中保存的是arr的指针。代码3中，第三句代码并不是直接修改了arr本身的值，而是给它引用了另一个对象，所以这是newArr并不能指向另一个引用，newArr中保存的也是arr的指针。
  1. JS语言特性二：作用域及变量提升

  • JS只有两种作用域，即全局作用域和函数作用域。函数拥有自己的作用域，而while、if、for等语句块则没有。JS中的全局变量是window对象的属性。

  • 在JS中，所有的var声明(不包括变量的初始值)都会被提升到作用域的顶部。

‘use strict’;

//Object.seal()可以限制对象的能力，已有的可写属性可以更新，但是属性不能删除，也不能转换成存取器，删除和添加属性都会报错。

greeting: ‘hello’,
innerObj: {}

Object.freeze(obj); //此时innerObj对象还可以修改
Object.freeze(obj.innerObj); //此时innerObj对象也被冻结

  1. JS不是基于类的语言，是一门基于原型的语言，一切对象都有原型。原型是对象的一个隐藏属性，可以用prototype（正式）和proto(非正式，在es6中成为正式）两种方法来引用。

  2. 原型链是用来解析对象成员值的两种途径之一，如果引用了foo.bar，那么JS引擎会在两个位置查找，foo本身以及他的原型链。
    bob和Person之间是一种行为委托的关系，Person对象生成的实例bob有自己的属性，但是没有自己的方法，他的方法委托给了Person对象。原型链就是一种委托机制，允许将行为委托给链上更远的一环来处理。

  3. 继承就是用Object.create()在父类型和子类型之间建立联系，子类型可以在继承到父类型所有属性的基础上新增，删除或者覆盖已有的属性。
    当在对象中定义好了创建实例的工厂函数时，用工厂函数创建的实例不能用instanceof来判断是否是对象的实例，可以用isPrototypeOf()方法来判断,也可以用getPrototypeOf()方法来获得某对象的原型。
