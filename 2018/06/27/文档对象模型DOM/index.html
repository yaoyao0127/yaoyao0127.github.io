文档对象模型DOM相关

发表于 2018-06-27 | 分类于 javascript
  1. DOM:Document Object Model，文档对象模型。
  2. 获取DOM元素的方法
  • DOM之间的相互关系：每个DOM元素都存放着一组指向他的直系父、子、兄弟的指针，可以通过这些指针的连接关系来访问DOM元素。
  • document.getElementById(“”)
  • document.getElementsByTagName(“”)：返回一组NodeList。
  • document.getElementsByClassName(“”)：返回一组HTMLCollection。
  • querySelector():传入一个合法的css选择器，就返回选择到的第一个元素的引用。
  • querySelectorAll():传入一个合法的css选择器，就返回选择到的所有元素的NodeList。
  1. NodeList和HTMLCollection
  • 同：都是DOM节点的集合；都是类数组对象，有length属性；都有item方法，可以通过item(index)或者item(id)来访问返回结果中的元素。
  • 异：HTMLCollection比NodeList多了一个namedItem方法；NodeList可以包含任何节点类型，HTMLCollection只包含元素节点。
  1. Node:是一个基础类型，document, element, text, comment, DocumentFragment等都继承自Node。
