JS基础：数据类型

发表于 2018-07-03 | 分类于 javascript

  1. JS共有5种基本数据类型：Number、String、Boolean、Null、Undefined和1种复杂数据类型Object，所有的值最终将是这6种数据类型之一。使用typeof操作符可以检查任何值的数据类型。

  2. Undefined 未定义

  • 该类型数据只有一个值，就是undefined。
  • 用var声明一个对象后还没有初始化的时候，这个对象的值就是undefined。
  • 引入undefined是为了区分空对象指针和未经初始化的变量。
  • typeof操作未被声明的变量和声明了但是未被初始化的变量时都会返回undefined。
  1. Null 空对象指针
  • 该类型数据只有一个值，就是null，表示一个空对象指针。
  • 如果定义的对象准备将来用于保存对象，那么最好把他初始化为null，在操作的时候只要检查该对象是否等于null，即可知道他是否已保存了一个对象的引用。
  • 只要意在保存对象的变量还没有真正保存变量，就应该明确地让该变量保存null值。
  1. Boolean
  • 该类型数据有两种取值，true和false。
  • 任何其他数据类型都有和true与false相对应的数据。
  • 将非Boolean类型数据转换为Boolean类型，可以使用转型函数Boolean()

  1. Number
  • 首位为0且后面的数字都小于8，按照8进制解析。严格模式下无效。
  • 首位为0x，后面的字符属于16进制数字，按照16进制解析。
  • 浮点数：小数点右面必须至少有一位数。
  • 保存浮点数的内存空间是整数的两倍，所以ECMAscript会不失时机地将浮点数转换为整数。
  1. String
  • 字符串创建之后的值无法改变，只能销毁以后用另一个包含新值得字符串填充该变量。
  • 数值、布尔值、对象和字符串都有toString()方法，null和undefined没有，该方法可以传递一个指定进制的参数。
  • 所有的数据类型都可以用转型函数String()转换为字符串。
  • “+”可以拼接字符串
  1. Object
  • 每个Object实例都具有的属性和方法：
    (1)constructor：保存着创建当前对象的函数
    (2)hasOwnProperty(propertyName)：检查给定的属性是否属于当前对象实例
    (3)isPrototypeOf(object)：检查传入的对象是否是当前对象的原型
    (4)propertyIsEnumerable(propertyName)：检查该属性是否可以用for-in枚举
    (5)toString()：返回字符串表示
    (6)toLocaleString()：返回字符串表示，与字符串环境的地区对应
    (7)valueOf()：返回字符串、数值或布尔值表示
